<Intro>
  The Hint component is deprecated and should no longer be used. Use the Gha (GitHub Alerts) component instead for displaying callouts and informational boxes with various severity levels.
</Intro>

> [!CAUTION]
> Deprecated, use [`Gha[keyword="NOTE"]`](gha) instead.

```md
<Hint>
  I'm a deprecated hint. Use `Gha` instead.
</Hint>
```

result:

<Hint>I'm a deprecated hint. Use `Gha` instead.</Hint>
