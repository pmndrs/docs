<Intro>
  The Grid component creates responsive grid layouts for organizing content. It automatically adjusts the number of columns based on screen size, making it ideal for displaying lists of items, cards, or any content that benefits from a structured grid layout.
</Intro>

```md
<Grid cols={2}>
  <li>A</li>
  <li>B</li>
  <li>C</li>
  ...
</Grid>
```

result:

<Grid cols={2}>
  <li>A</li>
  <li>B</li>
  <li>C</li>
  <li>D</li>
  <li>E</li>
  <li>F</li>
  <li>G</li>
</Grid>
