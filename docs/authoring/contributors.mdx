<Intro>
  The Contributors component displays avatars and profiles of GitHub repository contributors. It fetches contributor data from the GitHub API and presents them in a visual grid, acknowledging the people who have contributed to the project.
</Intro>

```md
<Contributors owner="pmndrs" repo="docs" />
```

> [!WARNING]
> [`CONTRIBUTORS_PAT`](introduction#configuration:~:text=CONTRIBUTORS_PAT) needs to be set. Otherwise, it will display John Doe.

result:

<Contributors owner="pmndrs" repo="docs" />
