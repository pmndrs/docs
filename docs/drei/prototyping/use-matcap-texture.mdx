---
title: useMatcapTexture
---

Loads matcap textures from this repository: [https://github.com/emmelleppi/matcaps](https://github.com/emmelleppi/matcaps)

(It is a fork of this repository:[ https://github.com/nidorx/matcaps](https://github.com/emmelleppi/matcaps))

```jsx
const [matcap, url] = useMatcapTexture(
 0, // index of the matcap texture https://github.com/emmelleppi/matcaps/blob/master/matcap-list.json
 1024 // size of the texture ( 64, 128, 256, 512, 1024 )
)

return (
 ...
 <meshMatcapMaterial matcap={matcap} />
 ...
)
```
